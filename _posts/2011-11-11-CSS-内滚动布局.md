~~~
$(function(){                           
 $("#filenameDiv").scroll(function() {      
    var offset = $(this).offset();      
    $("#filenameDiv-Btnes").offset(offset);  
  }) ;                             
});
~~~
{: .language-ruby}

这个会让 上方的排序栏目固定.
下面的 文件列表可以滚动.
但是滚动后内容会重叠....


 一般滚动是 整个页面的...

有时候要 某个div 滚动...



既然是内滚动条.
就必须干掉 浏览器原生的 html 滚动条
`	html { overflow: hidden; }`



然后让一个div滚动就可以了..
