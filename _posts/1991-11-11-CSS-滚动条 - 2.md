~~~
html,body { height: 100% }
#leftBig  { height: 100% ; width: 30%;  background-color:red; float:left;}
#rightBig { height: 100% ; width: 50%;  background-color:green; float:left;}
~~~
{: .language-ruby}

html body 高度设置 100%
然后两个div 都设置 float:left.
